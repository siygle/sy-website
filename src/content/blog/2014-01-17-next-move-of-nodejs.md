最近 nodejs 社群真是不平靜啊 XD

昨天看到公告之後有點震撼，不過驚嚇點不是在 nodejs 的專案領導者將由 [@izs](https://twitter.com/izs) 轉交給 [@tjfontaine](https://twitter.com/tjfontaine)，雖然也算是大事，不過其實算有跡可循，最近幾次的 Release 和 Github 上面都可以看出一些端倪。

相信多數人是被 [npm Inc.](http://npmjs.com) 嚇到的，頭頭換人的原因應該也是因為這個，看起來 Isaac 要將重心放在 npm 上面，然後也成立一間以提供 npm 相關服務的新公司。

雖然近期 npm 的穩定度讓我覺得 Isaac 能夠將全力放在改良 npm 上面一定是正面的影響，不過應該有不少人會擔心會不會因為商業化而有不好的影響。在目前還沒有更詳細的消息之前也不好說些什麼，不過我想一直都在從事 OSS 的 Isaac 以及本身就是以此發跡的 nodejs，應該是不會背離這個精神才是 ;p

倒是定位的問題讓我比較好奇，因為目前 npm 應該算是 nodejitsu 負責維護的，那麼往後 npm inc. 是單純提供服務而已嗎？

還是得先等進一步的消息出來再看看...

希望今年內可以看到 v1.0 出現啊!!
