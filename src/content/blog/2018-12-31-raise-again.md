終於在這 2018 的年底，打開了這塵封已久的角落（基本上算是整個重弄了），前陣子發現 [Ghost](https://github.com/TryGhost/Ghost) 已經默默走到 2.9 了（但是 app/plugin 依然沒有出現 XD），手賤升級之後就掛了，後來發現應該是 linode 給了升級，所以換了機器也一起讓在 [dokku](/use-dokku-setup-ghost/) 上的資料都變不見啦，幸好在這個 2018 的年代，基本上架個 blog 就跟 **Hello World** 差不多了呢，真是應該好好感謝眾多推動軟體進步的大大們 ;)

_不過真的要抱怨 Ghost 真的越改越爛，[中文輸入法](https://github.com/TryGhost/Ghost/issues/9801)完全無法在 2.x 之後的新編輯器上使用...0rz，除了花俏之外真的也看不出什麼大改進。_

### _停了好久_

嗯，也沒別的原因，就是**懶**。不過人生邁入另一個階段也是真的，所以沒辦法像之前那麼無聊老是寫廢文了...0rz

不過最近反思自己進度的動力也不如以往了，人生就是到高峰之後一步步的老化，各方面都是，真的要好好把握自己的黃金時期。

### [Node.js](http://nodejs.org)

還是愛它，但是應該會逐漸轉移興趣去別處。一腳誇進 OSS 應該是從自己入坑 Node.js 算起，_雖然 JS 漸漸取代 PHP 變成網路上的消遣目標 XD_，但是還是愛它的生氣跟社群的文化（當然只是追文跟 Github 上 issue 討論的感覺，還沒機會真的去參加國外的，希望有朝一日能夠圓夢 XD）

### chromium 統一世界?

其實當天看到 [新聞](https://github.com/MicrosoftEdge/MSEdge) 的時候本來想發個文，不過後來看到歐三大的文章 - [瀏覽器多樣性 Browser Diversity](https://blog.othree.net/log/2018/12/27/browser-diversity/) 已經針對這個新聞作了很詳盡的說明。不過老實說，我對這件事並沒有非常悲觀（當然個人還是支持多樣性的重要），這次 edge -\> chromium 的事件，跟 IE 之前獨大的問題應該不能等而視之。如果哪天 Google 真的要亂搞，說不定也是會出現像之前 Node.js/io.js 的情形。

### 2019，希望能重拾熱情再衝一次！Happy New Year!!
